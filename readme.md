## 背景
本次南工骁鹰电控框架更新计划于2021赛季结束后开始，至2022分区赛实战验证后基本固定。
鉴于C++版本基本框架已开源且已被广泛引用参考，以及后来人终将走上自己的长征路，故决定开源。
此次开源自用版C语言版本代码，以及一套麦轮步兵例程。2022后该框架及基于该框架的各机器人代码已由2023赛季电控组接管，其后修改与本次开源内容无关。
